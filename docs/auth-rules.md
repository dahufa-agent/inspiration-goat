# 注册与登录规则

## 一、注册规则

### 1.1 基本要求

| 字段 | 规则 | 示例 |
|------|------|------|
| **手机号** | 必须为11位，以1开头 | `18104962855` |
| **用户名** | 3-20位字母、数字或下划线 | `inspiration_goat` |
| **密码** | 至少6位 | `abc123` |
| **确认密码** | 必须与密码一致 | `abc123` |

### 1.2 验证流程

```
1. 输入手机号 → 格式校验（11位，以1开头）
2. 获取验证码 → 发送到手机（有效期10分钟）
3. 输入用户名 → 规则校验（3-20位字母、数字或下划线）
4. 输入密码 → 长度校验（至少6位）
5. 确认密码 → 与密码一致性校验
6. 点击注册 → 验证码校验 → 完成注册
```

### 1.3 验证规则优先级

1. **手机号格式**：`/^1\d{10}$/` - 必须是11位以1开头的数字
2. **用户名格式**：`/^[a-zA-Z0-9_]{3,20}$/` - 3-20位字母、数字或下划线
3. **密码长度**：最少6位字符
4. **密码确认**：两次输入必须完全一致
5. **验证码**：6位数字，有效期10分钟

### 1.4 错误提示

| 场景 | 错误提示 |
|------|----------|
| 手机号为空 | 请输入11位手机号 |
| 手机号格式错误 | 手机号格式不正确 |
| 用户名为空 | 所有字段都不能为空 |
| 用户名格式错误 | 用户名需要3-20位字母、数字或下划线 |
| 密码为空 | 所有字段都不能为空 |
| 密码太短 | 密码至少6位 |
| 密码不一致 | 两次密码输入不一致 |
| 验证码为空 | 请输入验证码 |
| 验证码错误 | 验证码无效或已过期 |
| 手机号已注册 | 该手机号已注册 |
| 用户名已占用 | 用户名已被占用 |

## 二、登录规则

### 2.1 基本要求

| 字段 | 规则 | 示例 |
|------|------|------|
| **用户名/手机号** | 手机号（11位）或用户名（3-20位字母、数字、下划线） | `18104962855` 或 `inspiration_goat` |
| **密码** | 至少6位 | `abc123` |

### 2.2 登录方式

支持**两种登录方式**（任选其一）：

1. **手机号登录**：输入注册时的手机号
2. **用户名登录**：输入注册时的用户名

### 2.3 验证流程

```
1. 输入用户名或手机号 → 支持两种格式
2. 输入密码 → 长度校验（至少6位）
3. 点击登录 → 验证用户名/手机号 + 密码
```

### 2.4 错误提示

| 场景 | 错误提示 |
|------|----------|
| 用户名为空 | 用户名和密码不能为空 |
| 密码为空 | 用户名和密码不能为空 |
| 用户名或密码错误 | 用户名或密码错误 |

### 3.1 永久会员识别

- 可以使用应用的基本功能

## 五、技术实现

### 5.1 前端规则校验

**注册表单校验**（客户端）：
```typescript
// 手机号：11位，以1开头
if (!/^1\d{10}$/.test(phone)) {
  return "手机号格式不正确";
}

// 用户名：3-20位字母、数字或下划线
if (!/^\w{3,20}$/.test(username)) {
  return "用户名需要3-20位字母、数字或下划线";
}

// 密码：至少6位
if (password.length < 6) {
  return "密码至少6位";
}

// 密码确认
if (password !== confirmPassword) {
  return "两次密码输入不一致";
}
```

### 5.2 后端规则校验

**注册接口** (`POST /api/v1/auth/register`)：
- 验证所有必填字段
- 验证手机号格式
- 验证用户名格式（3-20位字母、数字、下划线）
- 验证密码长度（至少6位）
- 验证验证码有效性
- 验证手机号唯一性
- 验证用户名唯一性
- 自动识别永久会员手机号

**登录接口** (`POST /api/v1/auth/login`)：
- 支持手机号或用户名登录
- 验证密码匹配
- 返回用户信息和会员状态

### 5.3 API 端点

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 发送验证码 | POST | `/api/v1/auth/send-code` | 发送注册验证码 |
| 注册 | POST | `/api/v1/auth/register` | 创建新账号 |
| 登录 | POST | `/api/v1/auth/login` | 用户登录 |
| 用户信息 | GET | `/api/v1/auth/user-info` | 获取当前用户信息 |
| 激活免费码 | POST | `/api/v1/auth/activate-free-code` | 激活会员 |
| 生成免费码 | POST | `/api/v1/auth/generate-free-code` | 生成邀请码（仅永久会员） |
